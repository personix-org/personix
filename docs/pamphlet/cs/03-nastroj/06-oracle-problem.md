# Oracle problém — propojení digitálního a fyzického světa

Pojem „oracle problém" je nejvíce v poslední době spojován se světem kryptoměn, blockchainů a decentralizovaných systémů. Obecně: jak zajistit, aby data vstupující do digitálního systému věrně odpovídala tomu, co se skutečně stalo ve fyzickém světě? Systém sám nedokáže ověřit, jestli někdo opravdu někomu ublížil nebo jestli byla smlouva skutečně porušena — vidí jen digitální záznamy.

Tady to řešíme přes autority — subjekty, které vsadí svou reputaci na to, že digitální záznam odpovídá fyzické skutečnosti. Čím vyšší reputace autority, tím věrohodnější záznam.
